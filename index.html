    Metrostroi-MiniMap.

    Metrostroi-MiniMap на Github это дополнение к моду Metrostroi (Garry's Mod).
    MiniMap Client (Standalone) альфа уже доступна!

    Загрузки(Downloads):

    (Сдесь вы можете скачать клиент и серверный модуль)

    • Windows Client
    • Windows Server Module
    Установка(Installation):
    • Windows Client:
      - Скачать, разархивировать, запустить.
      (Download, unzip, run.)
    • Windows Server Module:
      - Скачать, разархивировать 'gmsv_minimap_win32.dll' в 'Папка Сервера\garrysmod\lua\bin', запустить сервер.
      (Download, place 'gmsv_minimap_win32.dll' in 'Server Folder\garrysmod\lua\bin', run server.)
    Требования(Requirements):
    • Windows Client:
      - .NET Framework v4.5
    • Windows Server Module:
      - .NET Framework v4.6
      - Metrostroi-MiniMap
